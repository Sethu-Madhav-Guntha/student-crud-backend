# student-crud-backend
Description: Student Create, Retrieve, Update and Delete Operations at Backend

Created source folder for structuring models, routers, controllers, services and config.
Created .env file for fetching PORT and MONGODB_URI string connection from the developer.
Installed mongoose library for storing and fetching data.
 and dotenv library for importing data from .env file.
Created User model, route and controller for fetching and creating data at users collection.
